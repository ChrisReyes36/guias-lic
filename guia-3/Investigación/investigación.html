Clausuras en JavaScript


Closures


Una clausura o closure es una función que guarda referencias del estado adyacente (ámbito léxico). En otras palabras, una clausura permite acceder al ámbito de una función exterior desde una función interior. En JavaScript, las clausuras se crean cada vez que una función es creada.


Los closures, en castellano denominados cierres, cerraduras o clausuras, son una característica de algunos lenguajes entre los que se encuentra JavaScript.

El closure se genera cuando una función interna a otra función usa una variable local (o parámetro recibido) de la función externa.


Para entender las clausuras se tiene que entender las características básicas del lenguaje que las hacen posibles:
• Se puede definir una función dentro de otra.
• Las variables accesibles desde una función son las que ésta define y las que definen las funciones que la engloban.
• Mediante esa variable se puede ejecutar esa función en un ámbito diferente al que fue definida

Y es así cuando una función devuelve otra función, la segunda mantiene las referencias a las variables que fueron definidas en la primera.

Y el tener la capacidad de mantener las referencias, o englobarlas, o encerrarlas, es a lo que se llama clausura (español castellano) o closure en JavaScript.

¿Son los closures realmente útiles? Un closure permite asociar algunos datos con una función que opera sobre esos datos, entonces se puede utilizar un closure en cualquier lugar en el que normalmente pondría un objeto con un solo método.
